Introduction

Bulk Photo Uploader is a simple application running on Windows 10 with .Net 4.5 or above. It function as an all-in-one software for photo resizing, canvasing, processing, labeling and uploading to Facebook.

This document provide a basic user guide for Bulk Photo Uploader.

To use this program, you need to follow the follow steps:

Step 1
  • Login to Facebook
  • Change language if necessary

Step 2
  • Select where you want the picture upload to.

Step 3
  • Setup picture setting, including resize, canvas, overlay image/text, etc.

Step 4
  • Select picture to be upload

Step 5
  • Upload the picture
